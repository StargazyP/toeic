# TOEIC Vocabulary Learning App

A minimalist English vocabulary learning app — study 3 to 50 words a day with AI-powered practice.

## DEMO

- **Live / Store**: (Add your demo or store link here after release)
- Vocabulary flashcards, AI quiz, sentence practice, and composition

## Features

- **Vocabulary learning** — Choose 3–50 words per session, flashcard-style with TTS
- **AI quiz** — Fill-in-the-blank and Korean→English; AI-generated sentences; result history
- **AI sentence practice** — Example sentences and translation; practice mode (read Korean, write English)
- **AI composition** — Paragraph generation using studied words; compare with your own writing
- **My words** — Studied words, known/unknown, practice and quiz history

## Tech Stack

- **Frontend**: React Native, Expo, TypeScript
- **Backend**: Node.js, Express, TypeScript
- **Database**: MySQL
- **AI**: Local LLM (Ollama) with optional cloud fallback
