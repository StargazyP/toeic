## 1️⃣ 프로젝트 컨셉

- **핵심 아이디어**: 하루에 3~5개 단어/문장만 보여주고, 사용자가 반응만 기록  
- **극단적 미니멀 UX**: 설명, 점수, 스트릭, 광고, 메뉴, 설정 없음  
- **학습 방식**:
  - 즉각적 보상 ❌
  - 지연된 의미 제공 ⭕  
- **AI 역할**: 단어 선정 및 학습 큐레이션  
- **UX 철학**: 부담 없고, 열자마자 끝나는 경험  

---

## 2️⃣ 기술 스택

### 프론트엔드
- React Native + TypeScript  
- 화면 1개 구조 (홈)  
- 애니메이션: 단어 등장/사라짐  
- 폰트: 시스템/Inter/SF/Pretendard  
- 버튼: 최소 터치 영역 48px, 강조 없음  

### 백엔드
- Node.js / Spring  
- DB: Supabase / Firebase  
- 서버 호스팅: Fly.io / Render / Railway  
- 특징: 단순 API, MAU 1,000 기준 저비용 유지 가능  

### AI
- 배치 생성 방식: 하루 단어 설명/예문 사전 생성  
- AI 호출: 관리자 도구에서만  
- 실시간 LLM 호출 X  
- AI 토큰 비용: 월 약 1,500 ~ 4,000원

## 3️⃣ UI / UX 설계

### 화면 구조
1. 첫 실행: 하루 단어 수 선택 (3 / 5 / 7)  
2. 홈 화면: 단어 1개 + “알 것 같다 / 모르겠다” 선택  
3. 단어 선택 후: 단어 사라짐 → 다음 단어 등장  
4. 오늘 분량 끝: “오늘은 여기까지” → 홈 복귀  
5. 지연 보상 화면: 모르는 단어 의미 한 번만 노출  

### 핵심 UX 원칙
- 설명 없음  
- 점수, 스트릭, 레벨 없음  
- 광고/배너 없음  
- 선택 최소화  
- 생각하지 않아도 되는 경험 유지  

### 색상/타이포
- 배경: #FFFFFF 또는 #000000  
- 포인트 색상 1개 (#4F46E5 등)  
- 폰트: Regular~Medium  
- 버튼 강조 없음, 애니메이션 최소  

---

## 4️⃣ AI & 단어 선정 전략

- **AI 역할**: 단어 큐레이션, 하루 분량 선정  
- **AI 사용 방식**: 사전 배치 생성 → DB 저장, 사용자 요청 시 AI 호출 없음  
- **AI 토큰 비용**: 하루 20단어 × 200토큰 × 30일 = 120,000 tokens → 약 1,500 ~ 4,000원/월  
- **단어 반복 알고리즘**:
  - “알 것 같다” 단어 제외  
  - “모르겠다” 단어 우선 재노출  
  - 학습 통계 기반 선택  

---

## 5️⃣ 수익화 전략

- **원칙**:
  - 광고 없음  
  - 기능 잠금 없음  
  - UX 파괴 금지  
  - 후원: 능동적 + 조용히  

### 후원 UX
- 하루 학습 후 잠깐 노출  
- **문구 예시**:
이 앱은 광고 없이 유지됩니다.
하루에 보여지는 단어는 AI와 서버 비용이 필요합니다.
계속 존재했으면 좋겠다면, 조용히 후원할 수 있습니다.

- 버튼: 월 1,000~2,000원, 클릭 강조 없음, 건너뛰기 버튼 동등  
- 후원하지 않아도 기능 동일  

### 예상 수익 (MAU 1,000 기준)

| 항목 | 보수적 | 최대 |
|------|--------|-----|
| 후원 전환율 | 5% | 10% |
| 후원 금액 | 1,000원 | 2,000원 |
| 후원자 수 | 50 | 100 |
| 총 수익 | 50,000원 | 200,000원 |
| 서버 + AI 비용 | 30,000원 | 30,000원 |
| 순익 | 20,000원 | 170,000원 |

### 세금 (한국 기준)
- 면세사업자 가능 → VAT 0%  
- 소득세: 연간 2,040,000원 → 6% → 월 약 10,000원  
- 순수 손에 남는 금액: 월 약 160,000원

## 6️⃣ 초기 MAU 1,000 확보 전략

- **채널 전략**:
  - 자발적 공유 / 스크린샷 → 자연스러운 공유 유도  
  - 커뮤니티 침투: Reddit, Discord, 대학 커뮤니티  
  - 홍보 아닌 피드백 요청  
- **스토어 최적화 (ASO)**:
  - 키워드: “단어 3개”, “공부 안 하는 영어”  
  - 검색량 낮아도 정확한 타겟 공략  
- **출시 전 필수**:
  - 온보딩 10초 내 완료  
  - 첫 사용 완주 경험 확보  
  - 계정/로그인 X  
- **DAU/MAU 시뮬레이션**:
  - 일 방문자 80명 × 설치 전환율 25%  
  - 재방문율 40% → 1개월 내 MAU 1,000 달성 가능

## 7️⃣ 서버 & 운영 비용

- 서버 + DB: 0 ~ 27,000원/월 (MAU 1,000 기준)  
- AI 토큰: 1,500 ~ 4,000원/월  
- 총 운영비: 최대 30,000원/월  
- 비용 구조 선형 증가 아님, 소규모 MAU에서는 안정적  

---

## 8️⃣ 핵심 철학

- 극단적 미니멀: 사용자가 고민하지 않고 단순 행동만 하도록 설계  
- 조용한 후원: 돈을 강요하지 않고, 필요성을 이해하면 자연스레 발생  
- UX 우선: 광고, 스트릭, 랭킹, 점수 등 모든 방해 요소 제거  
- 지속가능성: MAU 1,000 이상이면 서버+AI 비용 자체 충당 가능

## 📱 극단적 미니멀 후원 UX (인앱 결제)

### 1️⃣ 하루 학습 완료 화면

오늘 학습 완료! 🎉
이 앱은 광고 없이 유지됩니다.
하루 단어 생성과 AI 학습이 필요합니다.

조용히 후원해주실래요? 👇
[후원하기] ← 클릭 1번
[건너뛰기] ← 클릭 1번


- 후원/건너뛰기 버튼 강조 최소  
- 텍스트 부담 없음, 선택권 균형 유지  

---

### 2️⃣ 후원 버튼 클릭 → 앱스토어 결제

결제 화면 (스토어 제공)
상품: TOEIC Minimal 후원
금액: 1,000원 / 월 구독
[결제하기] ← 클릭 1번
[취소] ← 클릭 1번


- Apple: StoreKit, Android: Google Play Billing  
- 클릭 1~2번으로 완료  
- 반복 결제 가능, 스토어가 환불/기록 처리

### 3️⃣ 결제 완료 팝업

오늘 하루 단어 학습에 도움을 주셨습니다.
[홈으로] ← 클릭 1번


- 짧고 단순  
- 홈 버튼만 제공 → 학습 루프 종료

### 5️⃣ React Native 구현 예시 (인앱 결제)

```ts
import * as RNIap from 'react-native-iap';
import { Platform, Alert } from 'react-native';

const itemSkus = Platform.select({
  ios: ['toeic_minimal_1000'],
  android: ['toeic_minimal_1000'],
});

async function purchaseItem() {
  try {
    await RNIap.requestPurchase(itemSkus[0]);
    Alert.alert(
      '감사합니다!',
      '오늘 하루 단어 학습에 도움을 주셨습니다.'
    );
  } catch (err) {
    console.warn(err);
  }
}
버튼 클릭 시 호출
